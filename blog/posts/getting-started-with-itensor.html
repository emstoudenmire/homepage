Getting Started with ITensor in Julia

January 15, 2025

If you're interested in tensor network methods but haven't tried coding them up, ITensor makes it surprisingly easy to get started. In this post I'll walk through installing the Julia version and running a basic DMRG calculation.

Installing ITensors.jl

First, make sure you have Julia installed (version 1.6 or later). Then open a Julia REPL and type: 

using Pkg
Pkg.add("ITensors")

That's it. Julia's package manager handles all the dependencies.

A first DMRG calculation

Let's find the ground state of a spin-1/2 Heisenberg chain. The Hamiltonian is:

H = ∑i Si · Si+1

In ITensor, we build this as a matrix product operator (MPO) using the OpSum system: 

using ITensors

N = 20
sites = siteinds("S=1/2", N)

os = OpSum()
for j in 1:(N - 1)
    os += "Sz", j, "Sz", j + 1
    os += 0.5, "S+", j, "S-", j + 1
    os += 0.5, "S-", j, "S+", j + 1
end
H = MPO(os, sites)

psi0 = random_mps(sites; linkdims=10)
sweeps = Sweeps(5)
setmaxdim!(sweeps, 10, 20, 100)
setcutoff!(sweeps, 1e-10)

energy, psi = dmrg(H, psi0, sweeps)

println("Ground state energy = ", energy)

This should converge to an energy per site close to the exact Bethe ansatz result of about −0.4432 for large systems. The key idea is that ITensor's named-index system handles all the bookkeeping of tensor contractions for you—you just specify the physics.

Next steps

From here you can explore measuring local observables, computing entanglement entropy, or trying different models. The ITensors.jl documentation has many more examples and tutorials.
